Overview

“맘으로(Mom-Ro)”는 엄마의 마음으로 육아의 방향을 제시하는 육아 정책 통합 플랫폼입니다.
바쁜 부모들이 소중한 육아 정책과 일상을 놓치지 않도록, 맞춤형 정책 안내부터 육아 정보, 기록, 커뮤니티, 커머스까지
육아에 필요한 모든 기능을 하나의 플랫폼에서 제공합니다. 육아 시, 맘으로와 함께하세요

Project Goal

맘으로 목표

따라서 바쁜 부모들이 언제 어디서나 육아 정책과 정보를 쉽게 확인할 수 있도록,
육아에 필요한 모든 기능을 하나로 통합한 ‘맘으로(Mom-Ro)’ 플랫폼을 만들어
정보 접근의 시간적·공간적 제약을 극복하고 육아 정책의 접근성을 높이고자 하였습니다.

Introduction about START MOM Team

스타트맘은 Start(시작)+Mom(엄마)를 합쳐 만든 팀명으로
엄마들의 새로운 시작을 응원한다는 마음에서 비롯된 팀명입니다.
저희는 ‘맘으로’라는 서비스를 통해, 엄마들이 새로운 시작에서 겪는 어려움을 덜어주고자 합니다.
